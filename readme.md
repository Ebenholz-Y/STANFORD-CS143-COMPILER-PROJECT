项目名称: COOL编译器实现
课程: Stanford CS143 - Compilers
完成时间: 2025/12/27
开发者: 姚涵芳

项目概述
实现了一个完整的编译器，将COOL语言(面向对象教学语言)编译为MIPS汇编代码。涵盖词法分析、语法分析、语义分析、代码生成等编译器核心阶段。

技术栈
编程语言: C++
工具: Flex, Bison, Make
目标架构: MIPS

快速开始
构建: make
测试: make test
编译: ./coolc [输入文件.cl]

项目结构
/src - 源代码
/include - 头文件
/test - 测试用例
/Makefile - 构建文件
